Projekt na podstawie:
https://www.youtube.com/watch?v=zuye-dYSkS0&list=PLzNdPxKZjud1IOZG8RWxA0U_497c7v2GP&index=2&t=16359s

Po uruchomieniu projektu w Visual Studio Code odpalamy terminal i instalujemy paczki: 
- npm install @vitejs/plugin-react
- npm install @vitejs/plugin-react-swc
a potem Split, żeby miec dwa okna. W jednym oknie wchodzimy w katalog frontend, a w drugim w beckend.

w terminalu frontend wpisujemy polecenia:
npm i react-router-dom axios react-icon
enter
npm install -d tailwindcss postcss autoprefixer
enter
npx tailwindcss init -p
enter

w terminalu backend:
npm init -y
npm i express mongoose dotenv cors bcrypt jsonwebtoken cookie-parser multer
enter
npm i nodemon
enter

uruchomienie projektu:
terminal backend - nodemon index.js
terminal frontend - npm run dev i klikamy link http://localhost:5173/ z Crt

Windows może blokować uruchamianie skryptów ze względów bezpieczeństwa, wtedy będą wyświetlane błędy i info o braku połączenia z serwerem, w takim wypadku przed uruchomieniem projektu w obu terminalach trzeba wpisać: Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass enter i potem komendy do uruchomienia
